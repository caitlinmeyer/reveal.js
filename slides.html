Reveal.js & Library Tutorials

Code4Lib Recap for SCOPA

September 29, 2017

Project Goals

  • Create access to comprehensive information without being too lengthy
  • Afford user agency and respect time
  • Accomodate varied learning styles
  • Leverage existing open resources

Technical Requirements

  • Compatible with campus LMS
  • Integrate assessment tool
  • Functional across devices
  • Modular construction
  • Multimedia capabilities

Reveal.js: a perfect fit

  • HTML presentation framework
  • Permits nested slides
  • Easy to write and work on collaboratively
  • Responsive design, houses multimedia content

Case Study: Zotero

Learning objectives:

  1. Create Zotero account
  2. Collect various types of resources
  3. Organize library into folders
  4. Generate a bibliography

Included Content

  • Videos demonstrating saving resources
  • Instructional GIFs for organization tips
  • Nine level top slides, 18 total
  • Integrated assessment tool

Publishing

  • Deployed in a LibGuide, made campus LMS module, WI+RE website
  • Available to other universities with liberal CC license
  • As of March, had been adopted by three other schools
  • Session duration over three minutes
